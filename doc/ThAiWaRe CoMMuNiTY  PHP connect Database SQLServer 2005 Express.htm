ผมจะใช้PHP 5 Run บน Apache 2.2 แล้วคราวนี้จะใช้ Database SQLServer 2005 Express( ใช้ user : Window Authentication) ทุกอย่างติดตั้งภายในเครื่องของผมเองครับ ครับ
ผมจะ connect มันได้ยังไงครับ

ถ้าผมใช้ mssql_connect
1. ต้องทำอย่างไรบ้างครับ ?(อ่านมาหลายเว็บแล้ว งงมาก ช่วยบอกเป็นลำดับทีนะคับ)
2. จะใช้ Servername ว่าอะไรครับ?
3 Username Password ที่ให้ในการ connect จะใช้อะไรครับ? (Window Authentication)

ช่วยคอบผมทีนะครับ ด่วนมากครับ

ที่ ผมทำครับ
$myServer = "localhost";
$myUser = "";
$myPass = "";
$myDB = "Decision";

$dbhandle = mssql_connect($myServer, $myUser, $myPass) or trigger_error(mssql_get_last_message(),E_USER_ERROR);
$selected = mssql_select_db($myDB, $dbhandle)or die("Couldn't open database $myDB");

ทำตามวิธีแบบนี้หมดแล้วครับไม่ได้เลย

เปิด file php.ini แล้ว เอาเครื่องหมาย ";" หน้า
;extension=php_mssql.dll ออก กลายเป็น
extension=php_mssql.dll
ตรวจดู ไฟล์ ntwdblib.dll ว่า มีอยู่ใน foder ของ php แล้วหรือไม่ ถ้ายังไม่มีก็ copy จาก แผ่น mssql มาได้

มันฟ้อง error แบบนี้ครับ
Warning: mssql_connect() [function.mssql-connect]: Unable to connect to server: localhost
